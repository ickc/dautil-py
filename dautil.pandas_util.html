dautil.pandas_util module¶

dautil.pandas_util.get_slice(df, name, no_levels=False, no_indexslice=False)[source]¶

Given name of an index from the MultiIndex from df, return the level this name is at, levels of all levels except this name is at, and an indexslice that slices all MultiIndex

Note: indexslice is of type list. Useful to be reassigned at level to another slice. When passed to pandas.DataFrame, need to be casted to tuple first.
