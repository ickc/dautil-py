    dautil.plot module¶

    dautil.plot.plot_camb(df, **kwargs)[source]¶

    plot DataFrame obtained by load_camb

    dautil.plot.plot_corr(df, vmin=0.0, mask=None, triangle=True, **kwargs)[source]¶

    Assume df is a correlation matrix if triangle, plot the lower-half triangle excluding diagonal only. mask will be passed to seaborn’s heatmap, data will not be shown in cells where mask is True.

    dautil.plot.plot_unique(df, col_select, col_plot, **kwargs)[source]¶

    plot the values of col_plot per unique value in col_select

    dautil.plot.plot_unique_index(df, idx_select, col_plot, **kwargs)[source]¶

    plot the values of col_plot per index from index level idx_select

    dautil.plot.plot_unique_index_binned(df, idx_select, col_plot, binwidth, **kwargs)[source]¶

    similar to plot_unique_index, but the values of index in level idx_select is binned instead Assumed df is sorted, else its behavior is undefined.

    Examples

    >>> plot_unique_index_binned(df_masked, 0, 'ampparams2_1', binwidth=pd.Timedelta('10 days'))
    dautil.plot.plot_x_y_complex(x, y, **kwargs)[source]¶

    x is a real array. y is a complex array.

    dautil.plot.save(f)[source]¶

    a decorator to add keyword filename to the function args the function f is expected to be a procedure that uses matplotlib the figure produced is saved to the path specified in filename dispatch according to the filename extension
